# Conventions

## Code
- TypeScript, Next.js App Router
- Minimal abstraction — ship fast for hackathon
- Token efficiency matters — choose tools that minimize LLM round-trips

## Quality Priority
1. Output video quality (cinematic, professional)
2. Token efficiency (fewer calls = faster + cheaper)
3. Agent-drivability (can an agent orchestrate the full pipeline?)
4. Code elegance (last priority)

## Hackathon Context
- ElevenHacks: must use Firecrawl Search + ElevenAgents
- Scoring: social posts (+50/platform), placement, virality, popularity
- The demo video IS the product — outputs are the submission
