# Clips MVP

## Problem
No tool lets you say "demo this feature on this website" and get a polished narrated video. Demo creation requires manual recording, editing, narration.

## Solution
Voice → Firecrawl (discover + scrape + automate) → agent-recorder (cinematic render) → ElevenLabs TTS (narrate) → final MP4.

## Acceptance Criteria

### Core Pipeline
- [ ] Firecrawl Search: given a feature description, find relevant URLs
- [ ] Firecrawl Scrape: convert URLs to markdown for LLM planning
- [ ] LLM Demo Planner: generate demo plan (actions + beats + narration script) from markdown
- [ ] Firecrawl Browser Sandbox: launch session, execute actions via Playwright/agent-browser
- [ ] CDP Recording: agent-recorder Rust binary captures via `--ws-endpoint <cdpUrl>`
- [ ] Remotion Render: cinematic post-processing (camera moves, beats, freezes, captions)
- [ ] ElevenLabs TTS: per-beat narration with `convertWithTimestamps`
- [ ] FFmpeg Merge: stitch audio + video → final.mp4

### Voice Interface
- [ ] ElevenAgent configured with `start_demo_generation` client tool
- [ ] Client tool params: `{ url, feature, audience, tone }`
- [ ] Agent asks follow-up questions when brief is unclear (plan mode)
- [ ] `sendContextualUpdate()` pushes pipeline progress to agent
- [ ] Embeddable in Next.js UI via `@elevenlabs/react`

### Web UI
- [ ] Landing page with voice agent embed
- [ ] Live View iframe showing browser automation in real-time
- [ ] Download link for finished video
- [ ] Demo gallery showing example outputs

### Hackathon Deliverables
- [ ] 3-4 demo videos generated by the tool (different personas)
- [ ] Main submission video (intro + live demo + outputs)
- [ ] Social clips for X, LinkedIn, TikTok/IG

## Technical Approach

### Pipeline Orchestration (`lib/pipeline/`)

```
orchestrator.ts
├── discover.ts    — Firecrawl Search API wrapper
├── understand.ts  — Firecrawl Scrape API wrapper
├── plan.ts        — LLM demo planner (Vercel AI SDK)
├── capture.ts     — Firecrawl Browser + agent-recorder CDP bridge
├── render.ts      — Remotion cinematic pipeline (calls agent-recorder tools)
├── narrate.ts     — ElevenLabs per-beat TTS + timestamp extraction
└── merge.ts       — FFmpeg audio/video merge
```

### Key Integration: CDP Bridge (`lib/pipeline/capture.ts`)

```
1. firecrawl.browser({ ttl: 120 }) → { id, cdpUrl, liveViewUrl }
2. Fork two parallel processes:
   a. agent-recorder --ws-endpoint <cdpUrl> --output raw.mp4 --fps 15
   b. firecrawl.browserExecute(id, { code: playwrightActions, language: "node" })
3. Wait for actions to complete
4. Stop recorder → raw.mp4 + sidecar.json
5. firecrawl.deleteBrowser(id)
```

### Key Integration: TTS Sync (`lib/pipeline/narrate.ts`)

```
For each beat in demoPlan.beats:
  1. Generate narration text (from beat.caption or LLM)
  2. elevenlabs.textToSpeech.convertWithTimestamps(voiceId, { text })
  3. Get audio buffer + character timestamps
  4. Adjust beat.holdMs to match audio duration
  5. Store word timestamps for Remotion subtitle overlay
Concat all beat audio → narration.mp3
```
